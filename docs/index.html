QUICK START
In this quick start. We are going to override a Symbol Instance’s Elements to display  different values when the Scene loads. We will change the Moon element width and Height And the Text box’s innerHTML. We will then drag to copy the Symbol on scene to add another instance of the same symbol and change its Property Values.

To add a new key/value click the plus symbol. for the Additional HTML Attributes
PropertyName
We will refer to the Key as the data-bindingName and the value as the PropertyName. The PropertyName is a Property of the element we want to override. The data-bindingName is an Arbitrary name we will use to link the new property values from the containing Symbols Attribute setting to an elements.  A data-bindingName can be whatever makes sense to you but must start with data-
